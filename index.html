# How it works (Python backend):
1. Haar Cascade detects faces
2. Face ROI → grayscale → resize to 48x48
3. Pre-trained CNN model predicts emotion
4. Draw bounding box + label + confidence

📋 Project Overview

  • 7 Emotion Classes - Trained on FER2013 dataset
  • Real-time Processing - 30+ FPS on standard webcam
  • Haar Cascade Face Detection - Fast & reliable
